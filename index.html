Recursion exercises:

This is a compilation of nine simple recursion exercises assigned by The Odin Project.
The following recursive functions can be run via the browser console.

Exercise 1: Sum all numbers.

Write a function called sumRange. It should take a number and return the sum of all numbers from 1 up to the number passed in.
Example: sumRange(3) returns 6.

Exercise 2: Power function.

Write a function called power which takes in a base and an exponent. If the exponent is 0, return 1.
Example: power(2, 4) returns 16.

Exercise 3: Calculate factorial.

Write a function called factorial which returns the factorial of a number.
Example: factorial(5) returns 120.

Exercise 4: Check all values in an array.

Write a function called all which accepts an array and a callback and returns true if every value in the array returns true when passed as a parameter to the callback function.
Example: all([1,2,9], (num) => {return num < 7}) returns false.

Exercise 5: Product of an array.

Write a function called productOfArray which takes in an array of numbers and returns the product of them all.
Example: productOfArray([1,2,3,10]) returns 60.

Exercise 6: Search JavaScript object.

Write a function called contains that searches for a value in a nested object. It returns true if the object contains that value.
Example:

const nestedObject = {
   data: {
     info: {
       stuff: {
         thing: {
           moreStuff: {
             magicNumber: 44,
             something: 'foo2'
           },
         },
       },
     },
   },
};

contains(nestedObject, 44) returns true, while contains(nestedObject, "foo") returns false.

Exercise 7: Parse a multi-dimensional array.

Write a function called totalIntegers which, given a multi-dimensional integer array, returns the amount of integers stored within.
Example: totalIntegers([[[5], 3], 0, 2, ['foo'], [], [4, [5, 6]]]) returns 7.

Exercise 8: Sum squares.

Write a function called sumSquares that, given an array, sums the squares of the numbers within the array and returns the result. The array could be multi-dimensional.
Example:
Both sumSquares([[1,2],3]) and sumSquares([1,2,3]) should return 14.

Exercise 9: Repeat number.

Write a function called replicate that takes two numbers as arguments and returns an array containing the first number repeated as many times as the second number argument's value.
If the times argument is negative, return an empty array.
Example:
replicate(5, 3) should return [5, 5, 5].
